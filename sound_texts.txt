The SiteAdmin module is used for EchoLink node or repeater administration. It allows a registered and authenticated user to send predefined commands over radio to the node or repeater for predefined actions by using DTMF codes.
You need to ask your sysop a personal login ID and password before start using this service.
To abort an operation  send a # or number sign.

HELP_LOGGED_IN_001

The following predefined commands are available and that are not defined by sysops:
send 0 for logged in help;
send 10 for powering off system;
send 69 to restart SvxLink;
send 73 to reboot the system;
send 77 to stop SvxLink;
send 87 to spell out OS, OS codename and OS version and
send 88 to say local IP address.
All of sysop defined commands are prefixed with 22.
All commands have to be ended with a # or number sign.
